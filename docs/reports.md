# 報告總覽

本系統產出以下主題報告，整合多個資料來源進行跨區域綜合分析。

## 報告類型

| 報告 | 頻率 | 說明 |
|------|------|------|
| [Rule Change Brief]({% link docs/Narrator/rule_change_brief/index.md %}) | 每週 | NIST 框架與標準變動簡報 |
| [AI Governance Landscape]({% link docs/Narrator/ai_governance_landscape/index.md %}) | 每月 | 跨區域 AI 治理動態 |
| [Supply Chain Security]({% link docs/Narrator/supply_chain_security/index.md %}) | 每週 | 全球供應鏈安全趨勢 |
| [Critical Infrastructure Resilience]({% link docs/Narrator/critical_infrastructure_resilience/index.md %}) | 每週 | 關鍵基礎設施韌性動態 |
| [Cybersecurity Compliance]({% link docs/Narrator/cybersecurity_compliance/index.md %}) | 每週 | 資安框架與法規變化 |

## 報告結構

每份報告包含：

1. **本期重點** — 3-5 項最重要的變動
2. **區域動態比較** — 美國 vs 歐盟
3. **責任變動追蹤** — 誰現在負責什麼
4. **義務與舉證要求** — 現在必須做什麼、證明什麼
5. **L5 Evolution Signals** — 長期趨勢觀察（系統推論）
6. **統計與資料來源** — 數據透明度
