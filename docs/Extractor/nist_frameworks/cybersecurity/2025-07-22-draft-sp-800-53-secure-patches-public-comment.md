## L1 — Rule Signal
- **rule_type**: draft
- **issuing_body**: NIST
- **document_id**: SP 800-53 Release 5.2.0
- **status**: public_comment

## L2 — Responsibility Structure
- **affected_roles**: Federal agencies (system administrators, security teams), Software developers, Organizations deploying patches, Security operations teams
- **shift_type**: expanded
- **shift_summary**: Organizations must now address software resiliency, developer testing, secure logging, least privilege controls, deployment management, software integrity validation, and delineate roles between organizations and developers for patch management

## L3 — Risk Domains
- Software supply chain security
- System integrity and reliability
- Patch deployment vulnerabilities
- Organizational accountability for secure updates

## L4 — Obligation & Evidence
- **new_obligations**:
  - Implement controls for secure and reliable deployment of patches and updates
  - Establish software resiliency measures and developer testing protocols
  - Deploy secure logging for patch activities with least privilege enforcement
  - Validate software integrity during update deployment
  - Document delineation of roles between organizations and software developers
  - Conduct root cause analysis and improvement processes for patch-related incidents
- **evidence_requirements**:
  - Documented deployment management procedures for updates
  - Software integrity validation records
  - Role and responsibility matrices between organizations and developers
  - Root cause analysis reports for patch deployment issues
  - Public comment submissions through NIST SP 800-53 Public Comment Site (deadline: August 5, 2025)
- **enforcement_signal**: mandatory

## Notes
This draft responds to Executive Order 14306 "Sustaining Select Efforts to Strengthen the Nation's Cybersecurity." The expedited two-week comment period (through August 5, 2025) and scheduled Release 5.2.0 by September 2, 2025 indicate urgency. Changes include one modified control enhancement, two new control enhancements, and six discussion updates. Contact: 800-53comments@list.nist.gov
