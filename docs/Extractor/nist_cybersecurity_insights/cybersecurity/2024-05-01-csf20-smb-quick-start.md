## Notes
本文介紹 CSF 2.0 SMB Quick Start Guide，定位為網路安全旅程的「入口道路」，專為「幾乎沒有網路安全計畫」的企業設計。六大功能框架：治理（建立並監控策略與政策）、識別（了解當前風險）、保護（實施風險管理防護）、偵測（發現並分析潛在攻擊）、回應（對已偵測事件採取行動）、恢復（復原受影響資產與業務）。每頁包含：考慮行動、入門指引（含規劃表格）、思考問題、延伸資源。提供靈活且非強制性框架、共同詞彙、與 MSSP 討論基礎。核心提醒：「網路安全是持續旅程」。Confidence 評為「中」是因為 Quick Start Guide 屬於實施輔助資源而非正式規範。
