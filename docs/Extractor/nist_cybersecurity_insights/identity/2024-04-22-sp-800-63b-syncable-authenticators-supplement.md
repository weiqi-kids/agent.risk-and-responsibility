## L1 — Rule Signal
- **rule_type**: new
- **issuing_body**: NIST Identity and Access Management Program
- **document_id**: NIST SP 800-63B Supplement (for incorporating syncable authenticators)
- **status**: final

## L2 — Responsibility Structure
- **affected_roles**: federal agencies, authentication system implementers, passkey adopters, enterprise and public-facing service providers
- **shift_type**: new
- **shift_summary**: NIST 首次為 SP 800-63B (Digital Identity Guidelines: Authentication and Lifecycle Management) 發布補充文件 (supplement)，提供聯邦機構在企業與公眾服務使用場景中採用 'syncable authenticators'（如 passkeys）的臨時指引

## L3 — Risk Domains
- Digital identity authentication
- Syncable authenticator security
- Passkey adoption
- Authentication lifecycle management

## L4 — Obligation & Evidence
- **new_obligations**: Federal agencies seeking to use syncable authenticators should follow interim guidance in SP 800-63B Supplement for both enterprise-facing and public-facing use cases
- **evidence_requirements**: Documentation of syncable authenticator implementation aligned with supplement guidance
- **enforcement_signal**: recommended

## Notes
本文以「補充劑」(supplements) 比喻，說明就像身體在變化環境中需要額外維生素（感冒季節的維C、冬季的維D），SP 800-63B 也需要補充文件協助適應變化。這是 SP 800-63B 首次發布 supplement，提供機構使用 syncable authenticators（例如 passkeys）的臨時指引。Description 足夠判斷文章主題，未使用 WebFetch。
