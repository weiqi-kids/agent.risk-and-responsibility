- **issuing_body**: NIST
- **document_id**: NIST Special Publication 800-181 (NICE Workforce Framework for Cybersecurity)

## L2 — Responsibility Structure

- **shift_summary**: NICE Framework 最新修訂新增工作角色、擴充能力領域至 11 個、更新任務知識技能（Task, Knowledge, Skill）描述，全面刷新資安人才框架

  - 雇主與教育機構應依據更新後的 NICE Framework 調整職能定義與訓練計畫
- **evidence_requirements**:
  - 無明確舉證要求
- **enforcement_signal**: recommended

## Notes
本文由 NICE Framework 首位全職經理發表，說明自 2020 年首次修訂以來的完整更新，包括修訂工作角色類別與角色（含一個新角色）、11 個新能力領域、更新的 Task/Knowledge/Skill 描述。
