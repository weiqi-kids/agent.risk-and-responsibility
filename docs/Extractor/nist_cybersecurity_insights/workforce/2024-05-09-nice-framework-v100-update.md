## L2 — Responsibility Structure

## L4 — Obligation & Evidence
- **new_obligations**:
  - K-12 教育應引導學生了解多元網路安全職涯
  - 高等教育與培訓機構應依 NICE Framework 提供技能對應的課程設計
  - 雇主應採用「技能型聘僱」（skills-based hiring）及多元人才管道
  - 組織應參考 11 個全新能力領域進行人才發展規劃
  - 關注新增的「內部威脅分析」工作角色
  - 持續關注未來工作角色新增（反映 AI 與網宇實體系統等新興技術需求）
- **evidence_requirements**:
  - 課程設計與 NICE Framework 的對應文件
  - 技能型聘僱的實施記錄
  - 能力領域的人才發展計畫
  - 新興技術相關的工作角色培訓記錄
- **enforcement_signal**: recommended

## Notes
本文公告 NICE Framework v1.0.0（2024-03-05 發布，NIST SP 800-181 rev.1）進行全面更新。修訂工作角色類別與工作角色（合併兩個重疊角色為「安全系統開發」、新增「內部威脅分析」）、新增 11 個能力領域（涵蓋跨工作角色的知識與技能群組）、更新 2,280 條 TKS 陳述（僅 139 條沿用自 2017 年版）。對人才培育影響：K-12 教育引導職涯、高等教育提供技能對應課程、雇主支援技能型聘僱及多元人才管道。未來將持續新增工作角色，反映 AI 與網宇實體系統等新興技術需求。此為「廣泛生態系共同推進網路安全人才的基礎」。Confidence 評為「高」是因為這是正式的 SP 修訂版發布。
