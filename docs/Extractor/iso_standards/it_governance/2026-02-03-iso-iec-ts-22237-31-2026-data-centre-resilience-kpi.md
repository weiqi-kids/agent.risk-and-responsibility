# ISO/IEC TS 22237-31:2026 — Information technology — Data centre facilities and infrastructures — Part 31: Key performance indicators for resilience

## L1: 基本資訊（必填）

**title**: Information technology — Data centre facilities and infrastructures — Part 31: Key performance indicators for resilience
**iso_refno**: ISO/IEC TS 22237-31:2026
**edition**: 2
**date**: 2026-02-03
**ics_code**: 35.020
**technical_committee**: ISO/IEC JTC 1/SC 39
**source_layer**: iso_standards
**original_content**: This document: a) defines metrics as key performance indicators (KPIs) for resilience, dependability, fault tolerance and availability tolerance for data centres; b) covers the data centre infrastructure (DCI) of power distribution and supply, and environmental control; c) can be referred to for covering further infrastructures, e.g. telecommunications cabling; d) defines the measurement and calculation of the KPIs and resilience levels (RLs); e) targets maintainability, recoverability and vulnerability; f) provides examples for calculating these KPIs for the purpose of analytical comparison of different DCIs. This document does not apply to IT equipment, cloud services, software or business applications.

## L2: 責任轉移判定（必填）

**shift_type**: clarified
**enforcement_signal**: recommended
**rule_type**: Published
**affected_party**: Data centre operators, infrastructure managers, resilience engineers
**explanation**: 此為第 2 版技術規範，定義資料中心韌性、可靠性、容錯和可用性容忍度的關鍵績效指標（KPIs）。涵蓋電力配送供應和環境控制的資料中心基礎設施（DCI），也可擴展至電信布線等其他基礎設施。定義 KPIs 和韌性等級（RLs）的測量和計算，針對可維護性、可恢復性和脆弱性。60 頁的內容提供了不同 DCI 的分析比較範例。不適用於 IT 設備、雲端服務、軟體或業務應用。

## L3: 歷史脈絡（選填）

**replaces**: ISO/IEC TS 22237-31:2023 (standard/83496)
**context**: 資料中心韌性是確保數位服務連續性的關鍵，但韌性衡量標準往往不一致。本技術規範屬於 ISO/IEC 22237 系列（資料中心設施和基礎設施），提供了標準化的韌性 KPIs，支援資料中心設計、營運和風險管理決策。
