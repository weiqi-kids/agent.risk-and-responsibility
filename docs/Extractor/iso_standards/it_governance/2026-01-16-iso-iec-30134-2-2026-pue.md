# ISO/IEC 30134-2:2026 — Information technology — Data centres key performance indicators — Part 2: Power usage effectiveness (PUE)

## L1: 基本資訊（必填）

**title**: Information technology — Data centres key performance indicators — Part 2: Power usage effectiveness (PUE)
**iso_refno**: ISO/IEC 30134-2:2026
**edition**: 2
**date**: 2026-01-16
**ics_code**: 27.015, 35.020
**technical_committee**: ISO/IEC JTC 1/SC 39
**source_layer**: iso_standards
**original_content**: This document specifies power usage effectiveness (PUE) as a key performance indicator (KPI) to quantify the efficient use of energy. This document: a) defines the PUE of a data centre; b) introduces PUE measurement categories; c) describes the relationship of this KPI to a data centre's infrastructure, information technology equipment and information technology operations; d) defines the measurement, the calculation and the reporting of the parameter; e) provides information on the correct interpretation of the PUE. PUE derivatives are described in Clause 9.

## L2: 責任轉移判定（必填）

**shift_type**: clarified
**enforcement_signal**: recommended
**rule_type**: Published
**affected_party**: Data centre operators, cloud service providers, facility managers
**explanation**: 此為第 2 版標準，規範電力使用效率（PUE）作為量化能源使用效率的關鍵績效指標（KPI）。定義了資料中心的 PUE、引入測量類別、描述與基礎設施/IT 設備/IT 營運的關係、規定測量/計算/報告方法，並提供正確詮釋 PUE 的資訊。第 9 條描述了 PUE 衍生指標。

## L3: 歷史脈絡（選填）

**replaces**: ISO/IEC 30134-2:2016 (standard/63451), ISO/IEC TS 30134-2:2016 (standard/72702)
**context**: PUE 是資料中心能源效率的業界標準指標，由 Green Grid 組織提出後被廣泛採用。本次第 2 版更新反映了資料中心技術的演進（如液冷、AI 工作負載）和能源效率測量方法的精進，支援淨零碳目標下的資料中心永續營運。
