# ISO/IEC 22121-3:2025 — Information technology — Virtual keyboards user interfaces — Part 3: Virtual keyboards interactions

## L1: 基本資訊（必填）

**title**: Information technology — Virtual keyboards user interfaces — Part 3: Virtual keyboards interactions
**iso_refno**: ISO/IEC 22121-3:2025
**edition**: 1
**date**: 2025-12-17
**ics_code**: 35.180
**technical_committee**: ISO/IEC JTC 1/SC 35
**source_layer**: iso_standards
**original_content**: This document provides guidelines on assistive technologies and functionalities that are to be included in accessible virtual keyboards. This document underlines which kinds of interaction modes can be embedded, which kinds of assistive technologies can be easily bound to virtual keyboards, and which kinds of functionalities are to be taken into account within virtual keyboards. This document does not apply to physical keyboards that use real physical keys or adaptable keys, which can be customized to user needs, for example with LCD display.

## L2: 責任轉移判定（必填）

**shift_type**: new
**enforcement_signal**: recommended
**rule_type**: Published
**affected_party**: Virtual keyboard developers, OS vendors, assistive technology providers
**explanation**: 此為新發布的國際標準（第 1 版），提供無障礙虛擬鍵盤應包含的輔助技術和功能指引。明確了可嵌入的互動模式、可輕鬆綁定的輔助技術類型，以及虛擬鍵盤應考量的功能。不適用於使用實體按鍵或可自訂適配按鍵的物理鍵盤。

## L3: 歷史脈絡（選填）

**replaces**: N/A
**context**: 虛擬鍵盤（螢幕鍵盤）已成為觸控裝置的標準輸入方式，但其無障礙設計往往不足。本標準回應了身心障礙使用者對可及虛擬鍵盤的需求，屬於 ISO/IEC 22121 系列的第三部分，補足了虛擬鍵盤互動和輔助技術整合的標準化空白。
