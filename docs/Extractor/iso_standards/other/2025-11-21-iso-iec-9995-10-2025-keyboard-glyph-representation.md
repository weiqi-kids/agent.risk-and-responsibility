# ISO/IEC 9995-10:2025 — Information technology — Keyboard layouts for text and office systems — Part 10: Conventional symbols and methods to represent graphic characters not uniquely recognizable by their glyph on keyboards and in documentation

## L1: 基本資訊（必填）

**title**: Information technology — Keyboard layouts for text and office systems — Part 10: Conventional symbols and methods to represent graphic characters not uniquely recognizable by their glyph on keyboards and in documentation
**iso_refno**: ISO/IEC 9995-10:2025
**edition**: 2
**date**: 2025-11-21
**ics_code**: 35.180
**technical_committee**: ISO/IEC JTC 1/SC 35
**source_layer**: iso_standards
**original_content**: Within the general scope described in ISO/IEC 9995-1, this document specifies symbols and methods to distinguish graphic characters which share the same glyph or are represented by similar glyphs on keyboards (including virtual keyboards) and in documentation. Each of these symbols is intended to be considered as universal and non-language related equivalent of names for the function they represent.

## L2: 責任轉移判定（必填）

**shift_type**: clarified
**enforcement_signal**: recommended
**rule_type**: Published
**affected_party**: Keyboard documentation writers, technical writers, UI designers
**explanation**: 此為第 2 版標準，在 ISO/IEC 9995-1 框架下規範符號和方法，以區分在鍵盤（包括虛擬鍵盤）和文件中共享相同字形或由相似字形表示的圖形字符。每個符號旨在作為其所代表功能的通用、非語言相關的名稱等效物。解決了不可見字符（如空格、換行）和外觀相似字符的標示問題。

**context**: 某些字符在鍵盤上難以區分（如不同寬度的空格、不同類型的破折號），影響技術文件的準確性和鍵盤標示的清晰度。本標準提供了標準化的表示方法。
