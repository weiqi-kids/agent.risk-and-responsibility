# ISO/IEC 23090-16:2025 — Information technology — Coded representation of immersive media — Part 16: Reference software for versatile video coding

## L1: 基本資訊（必填）

**title**: Information technology — Coded representation of immersive media — Part 16: Reference software for versatile video coding
**iso_refno**: ISO/IEC 23090-16:2025
**edition**: 2
**date**: 2025-11-20
**ics_code**: 35.040.40
**technical_committee**: ISO/IEC JTC 1/SC 29
**source_layer**: iso_standards
**original_content**: This document provides reference software for Rec. ITU-T H.266 | ISO/IEC 23090-3. The reference software includes both encoder and decoder functionality. Reference software is useful in aiding users of a video coding standard to establish and test conformance and interoperability, and to educate users and demonstrate the capabilities of the standard. For these purposes, the accompanying software is provided as an aid for the study and implementation of Rec. ITU-T H.266 | ISO/IEC 23090-3.
