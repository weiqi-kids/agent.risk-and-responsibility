**標準範圍**: This document specifies the abstract class of a media thing (MThing), a basic component to construct the Internet of media things. The MThing class contains the basic APIs to:—     discover other MThing(s) in the network;—     connect/disconnect MThing(s); and,—     support transactions (e.g. payments) using media tokens between MThings.
