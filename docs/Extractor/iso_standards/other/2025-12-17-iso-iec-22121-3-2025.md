**標準範圍**: This document provides guidelines on assistive technologies and functionalities that are to be included in accessible virtual keyboards. This document underlines which kinds of interaction modes can be embedded, which kinds of assistive technologies can be easily bound to virtual keyboards, and which kinds of functionalities are to be taken into account within virtual keyboards.This document does not apply to physical keyboards that use real physical keys or adaptable keys, which can be customized to user needs, for example with LCD display.
