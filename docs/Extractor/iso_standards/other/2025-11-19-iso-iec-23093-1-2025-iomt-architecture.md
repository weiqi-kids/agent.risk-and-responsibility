# ISO/IEC 23093-1:2025 — Information technology — Internet of media things — Part 1: Architecture

## L1: 基本資訊（必填）

**title**: Information technology — Internet of media things — Part 1: Architecture
**iso_refno**: ISO/IEC 23093-1:2025
**edition**: 3
**date**: 2025-11-19
**ics_code**: 35.040.40
**technical_committee**: ISO/IEC JTC 1/SC 29
**source_layer**: iso_standards
**original_content**: This document describes the architecture of systems for the internet of media things. It also includes a comprehensive set of use cases that can be deployed on such an architecture.

## L2: 責任轉移判定（必填）

**shift_type**: clarified
**enforcement_signal**: recommended
**rule_type**: Published
**affected_party**: IoT developers, media streaming providers, edge computing vendors
**explanation**: 此為第 3 版標準，描述媒體物聯網（Internet of Media Things, IoMT）系統的架構。包含一組可在此架構上部署的完整使用案例集。IoMT 結合了物聯網和媒體技術，支援智慧媒體裝置之間的協作和媒體內容的分散式處理。

## L3: 歷史脈絡（選填）

**replaces**: ISO/IEC 23093-1:2020 (standard/81586)
**context**: 媒體物聯網是物聯網和多媒體技術融合的新興領域，涵蓋智慧相機、無人機、穿戴裝置等能捕獲、處理和傳輸媒體的智慧裝置。本次第 3 版更新反映了邊緣運算、5G 網路、AI 媒體處理等技術的進展。
