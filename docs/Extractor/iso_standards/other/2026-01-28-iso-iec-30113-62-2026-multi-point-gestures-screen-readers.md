# ISO/IEC 30113-62:2026 — Information technology — Gesture-based interfaces across devices and methods — Part 62: Multi-point gestures for screen readers

## L1: 基本資訊（必填）

**title**: Information technology — Gesture-based interfaces across devices and methods — Part 62: Multi-point gestures for screen readers
**iso_refno**: ISO/IEC 30113-62:2026
**edition**: 1
**date**: 2026-01-28
**ics_code**: 35.240.20
**technical_committee**: ISO/IEC JTC 1/SC 35
**source_layer**: iso_standards
**original_content**: This document defines multi-point gestures for screen readers. It specifies movements for clear and classified multi-point gestures recognized by the screen readers. It describes multi-point gestures performed by multiple point of interest (POI) for the screen readers. NOTE: Multiple POI can be manipulated by using multiple objects such as fingertips, etc.

## L2: 責任轉移判定（必填）

**shift_type**: new
**enforcement_signal**: recommended
**rule_type**: Published
**affected_party**: Screen reader developers, assistive technology providers, OS vendors
**explanation**: 此為新發布的國際標準（第 1 版），定義螢幕閱讀器的多點手勢。規範了螢幕閱讀器識別的清晰且分類的多點手勢動作，描述了透過多個興趣點（POI）執行的多點手勢（如多指觸控）。填補了視障輔助技術手勢介面標準化的空白。

## L3: 歷史脈絡（選填）

**replaces**: N/A
**context**: 觸控介面普及後，視障使用者透過螢幕閱讀器使用觸控裝置的需求大增。然而不同平台的螢幕閱讀器手勢不一致，造成學習障礙。本標準屬於 ISO/IEC 30113 系列（跨裝置手勢介面），專注於螢幕閱讀器的多點手勢標準化。
