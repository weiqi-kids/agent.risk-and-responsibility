# ISO/IEC 4933:2026 — Information technology — User interfaces — Unifying input actions across devices

## L1: 基本資訊（必填）

**title**: Information technology — User interfaces — Unifying input actions across devices
**iso_refno**: ISO/IEC 4933:2026
**edition**: 1
**date**: 2026-01-26
**ics_code**: 35.180
**technical_committee**: ISO/IEC JTC 1/SC 35
**source_layer**: iso_standards
**original_content**: This document specifies a unification module for input actions, including its framework, requirements and recommendations, for consistently accessing the operations across multiple types of devices. This document recognizes the input actions as user behaviours that are commonly used to manipulate system or application operations (functions) through user interfaces. It also identifies the need and use cases of unifying input actions across devices. This document does not specify parameter settings and details of the input actions. This document is applicable to all types of organizations.

## L2: 責任轉移判定（必填）

**shift_type**: new
**enforcement_signal**: recommended
**rule_type**: Published
**affected_party**: Device manufacturers, UI/UX designers, software developers
**explanation**: 此為新發布的國際標準（第 1 版），規範跨裝置輸入動作的統一模組，確保使用者在不同裝置上能以一致的方式存取操作。標準辨識了使用者透過介面操控系統或應用程式的常見行為模式，並建立統一框架。適用於所有類型的組織。

## L3: 歷史脈絡（選填）

**replaces**: N/A
**context**: 在多裝置生態系統日益普及的背景下（智慧型手機、平板、筆電、穿戴裝置、IoT 裝置等），使用者經常需要在不同裝置間切換，造成學習成本和一致性問題。本標準回應了跨裝置使用體驗一致性的需求，為輸入動作的標準化提供了框架。
