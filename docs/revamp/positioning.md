# 品牌定位文件

> 分析日期：2026-02-19
> 網站：https://risk.weiqi.kids/

---

## 1. 核心資訊

| 項目 | 內容 |
|------|------|
| **網站名稱** | Risk & Responsibility Intelligence |
| **一句話描述** | 全球風險與責任情報系統——自動追蹤框架、法規與產業規則變動 |
| **核心價值主張** | 以 AI 自動化方式，從 NIST、EUR-Lex、CISA、ISO 等權威來源萃取結構化情報，協助專業人員每週快速掌握責任歸屬的演化趨勢 |

---

## 2. 目標受眾

### 主要受眾 (Primary)

| 項目 | 描述 |
|------|------|
| **是誰** | 企業 CISO、資安主管、法遵/合規主管 |
| **目標/需求** | 快速掌握每週法規變動，評估對組織的影響，確保合規 |
| **痛點** | 法規來源分散（NIST、EU、ISO 等）、人工追蹤耗時、難以判斷優先級 |
| **使用場景** | 每週一早上檢視報告，準備向管理層彙報；合規審查前查閱歷史變動 |

### 次要受眾 (Secondary)

| 項目 | 描述 |
|------|------|
| **是誰** | AI/ML 產品團隊、採購/供應商管理人員、OT/ICS 安全人員 |
| **目標/需求** | 了解特定領域（AI 治理、供應鏈、關鍵基礎設施）的最新要求 |
| **痛點** | 專業領域法規變化快，缺乏整合性追蹤工具 |
| **使用場景** | 新產品上線前的合規檢查、供應商評估、基礎設施安全規劃 |

---

## 3. 差異化定位

| 項目 | 內容 |
|------|------|
| **與競品的差異** | 完全自動化（AI 萃取 + 結構化輸出）、公開透明（原始碼開源）、免費使用 |
| **獨特優勢** | 週期性更新（每週報告）、多維度分類（rule_type、enforcement_signal）、責任變動追蹤（shift_type、affected_roles）|
| **不做什麼** | 不提供法律建議、不取代人工專家判斷、不處理機密或付費資料源 |
