## 6. 風險與依賴

### 風險

| 風險 | 影響 | 緩解措施 |
|------|------|----------|
| TBT 優化效果有限 | 無法達到 < 200ms | 考慮移除非必要 JS |
| PDF 生成複雜 | 延遲交付 | 先用手動轉檔，後續自動化 |
| Jekyll 限制 | RSS 功能受限 | 使用 jekyll-feed 插件 |

### 依賴

| 依賴項目 | 影響範圍 | 處理方式 |
|----------|----------|----------|
| Jekyll 插件 | RSS Feed | 確認 GitHub Pages 支援 |
| 報告模板修改 | 所有報告 | 需更新 Mode CLAUDE.md |
