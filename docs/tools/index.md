# 互動工具

本站提供實用的合規與風險評估工具，協助您快速了解組織的安全成熟度。

## 可用工具

### [合規自評工具](/tools/compliance-assessment/)

多框架整合的互動式合規檢核，包含：

- **NIST CSF 2.0** — 網路安全框架成熟度自評
- **GDPR** — 歐盟資料保護規範合規檢核
- **供應鏈安全** — SSDF、SBOM、第三方風險評估

**功能特色**：
- 即時計分與成熟度等級判定
- 答案自動儲存（瀏覽器 localStorage）
- 可下載 Markdown 格式報告

---

## 工具使用說明

1. 選擇您要進行的評估框架
2. 根據組織實際情況回答問題
3. 完成後查看分數與建議
4. 可下載報告供內部參考

> **免責聲明**：本工具僅供參考，不構成法律或專業建議。實際合規狀況請諮詢專業顧問。
